Para el curso de tensorflow se desarrollaron los archivos subidos al repositorio de GitHub.
Para que los archivos funcionen de manera correcta es necesario tener en cuenta lo siguiente:

-Los archivos se desarrollaron en Elementary OS

-Instalar anaconda segun su sistema operativo y para python 3

-Crear en entorno de trabajo

-Instalar las librerias de TensorFlow 1, Python 3.5, jupyter, keras, numpy, matplotlib y pandas

-Abrir jupyter notebook y buscar la ruta de los archivos

-Destacar que en elgunos es necesario descargar los archivos que se utilizan para los ejemplos de ser necesario cambiar la ruta dode se encuentre el archivp
